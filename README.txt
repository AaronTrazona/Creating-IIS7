
*Installing IIS7

1.) Start > Control Panel > Programs and Features > Turn Windows features on or off.
2.) In Server Manager > Click Roles > Add Roles.
3.) In Add Roles Wizard > Click Server Roles > Check Web Server (IIS) > Next > Next > Click Application Development > Install.
4.) Close the Server Manager.
